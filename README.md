## Setup

```bash
alias config='/usr/bin/git --git-dir=$HOME/.cfg/ --work-tree=$HOME'
echo ".cfg" >> .gitignore
git clone --bare git@github.com:khoi/dotfiles.git $HOME/.cfg  
config checkout
```
